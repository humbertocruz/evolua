# @evolua/web

App web principal do Evolu[a].

Neste momento, este app funciona como:
- base do cockpit `/evolua`
- embrião natural do SaaS do Evolu[a]
- referência visual e técnica para a experiência web

## Direção

A arquitetura atual do produto é:
- Evolu[a] como SaaS
- modelo vivendo na nuvem
- runtime local leve para apps Next.js do usuário

Por isso, este app não deve ser tratado como "pacote de integração Next do usuário".
Ele é o **produto web do Evolu[a]**.

npm run dev
