# @evolua/next (future)

Placeholder para o futuro pacote de integração do Evolu[a] com Next.js.

A ideia atual é que este pacote ofereça uma camada leve para apps Next.js do usuário:
- resolver rotas
- autenticar com o Evolu[a] Cloud
- buscar páginas/componentes remotos
- renderizar localmente o que vier do modelo

Ainda não é a hora de materializar este pacote por completo.
Ele existe aqui apenas para marcar a direção do monorepo.
