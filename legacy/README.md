# Legacy

Esta pasta guarda protótipos, demos e material de fases anteriores do Evolu[a].

Ela existe por três motivos:
- preservar histórico útil
- permitir reaproveitamento pontual
- evitar que ideias antigas continuem misturadas à arquitetura oficial atual

## Regra prática

- use `ideas/current-direction.md` como fonte oficial do rumo atual
- trate o conteúdo daqui como referência histórica, não como base principal do produto
